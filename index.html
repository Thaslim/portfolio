Prototype of Splitwise Application using MERN stack

In this project, developed a fully functional MERN stack prototype of Splitwise Application and deployed it on AWS EC2. Use Passport.js and JWT for authentication. When you are dealing with Audio data , it is hard to find sophisticated libraries in the programming languages supported by the mobile deployment platform such as Android. In this article I describe how to reuse the code in python inside your Android application with the help of Chaquopy! (Python SDK for Android)
